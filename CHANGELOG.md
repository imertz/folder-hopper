# Changelog

All notable changes to the "Folder Hopper" extension will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-01-30

### Added

- Initial release
- Go to Folder command with searchable Quick Pick
- Go to Folder (New Window) command
- Open Parent Folder command
- Open Parent Folder (New Window) command
- Context menu integration in Explorer
- Configurable scan depth
- Configurable exclude patterns
- Option to show/hide hidden folders
- Alphabetical sorting option
- Multi-root workspace support
- Async folder scanning with cancellation support
